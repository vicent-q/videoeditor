# videoeditor
云剪辑是一个云端一体的产品，包括前端组件/页面与后端服务两大部分。 

本仓库为云剪辑组件，基于React开发，支持多轨编辑，视频剪切、合并，画面裁剪，音频分离，转场特效，图片贴纸，文本字幕叠加等多项功能。
